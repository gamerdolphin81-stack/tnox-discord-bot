# Privacy Policy for TNOX

**Effective Date:** July 23, 2026

Thank you for using TNOX.

This Privacy Policy explains what information TNOX collects, how it is used, and how it is protected.

## Information We Collect

TNOX may collect and store the following information required for its operation:

- Discord Server IDs
- Discord User IDs
- Discord Channel IDs
- Role IDs
- Message IDs (only when required for bot features)
- Server configuration and settings
- Command usage information
- Moderation logs and configuration

TNOX does **not** collect Discord passwords, payment information, or personal messages unless explicitly required by a feature enabled by the server administrator.

## How We Use Information

Collected information is used to:

- Provide bot functionality
- Store server-specific settings
- Execute moderation features
- Improve reliability and performance
- Diagnose bugs and errors

## Data Sharing

TNOX does not sell, rent, or share collected information with third parties except when required by law.

## Data Retention

Server configuration data is stored only while necessary for bot functionality.

When TNOX is removed from a server, related server configuration may be deleted.

## Security

Reasonable efforts are made to protect stored information. However, no online service can guarantee complete security.

## Children's Privacy

TNOX is intended for users who comply with Discord's Terms of Service.

## Changes

This Privacy Policy may be updated at any time. Continued use of TNOX after changes constitutes acceptance of the updated policy.
